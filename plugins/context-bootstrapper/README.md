# context-bootstrapper

Bootstraps durable AI context from incomplete and messy human inputs.

This plugin is designed for the early phase of project formation where information is fragmented across:

- screenshots
- whiteboards
- conversations
- repositories
- TODO lists
- rough architecture sketches
- emotional/user intent
- partial workflows
- voice notes
- scattered documents

The skill helps Claude:

1. Extract durable concepts
2. Identify entities and relationships
3. Separate facts from hypotheses
4. Detect open questions and ambiguity
5. Propose missing structure
6. Build a living context graph
7. Suggest next bootstrap actions

## Skills

### `bootstrap-context`

Turns messy project information into structured, reusable context suitable for long-term AI collaboration and orchestration.

**Use when:**
- Starting a new project
- Reconstructing a forgotten design
- Turning screenshots into architecture
- Building AI memory systems
- Organizing fragmented technical ideas
- Creating durable agent context
